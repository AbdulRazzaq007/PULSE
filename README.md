# Pulse Review Scraper

A Python-based script to scrape product reviews from **G2** and **Capterra** for a given company and time period. The scraper outputs reviews in JSON format for further analysis or reporting.

---

## Features

- Scrape reviews from **G2** and **Capterra**.
- Filter reviews by **company name** and **date range**.
- Option to choose **specific sources** or scrape **all sources** at once.
- Save results in a **JSON file** for easy use.

---

## Project Structure

- `scraper.py` - Main script. Run with **Python 3.8+**.  
- `requirements.txt` - Required Python packages.  
- `sample_output.json` - Example JSON file generated by the scraper.
